　　前　言
　　电磁式振动台是一种精密的振动试验系统。每台系统都有最大动作特性。从事振动试验的技术人员，在做动态试验以前，必须根据试验参数和试件质量，作认真的数据计算和分析，以便安全有效地做好振动试验。为了便于论述，本文引用了我国苏州试验仪器厂研制的D-1000-5振动台和KZS-1数字式振动控制系统中的一些参数和技术指标。例如图1所示，是我国苏州试验仪器厂生产的D-1000-5电磁振动台系统的最大动作特性曲线。下面的“最大值求法”就是针对以上问题做出的技术性数据处理。


图1　D-1000-5最大动作特性曲线
　　另外，我们在实际工作中，常常遇到这样的问题，试验参量常常以隐含形式给出。如，已知2g定加速度与8mm定位移交越时的频率是多少?这样就必须求出交越频率点。这样便把物理问题归结为数学问题加以解决。作者以C语言写出计算程序，供大家简便而快捷地进行数据分析。

2　关于最大值的求法
2.1　最大负载质量的计算
　　使用电磁振动台系统进行正弦振动试验时，夹具和试件的质量是我们必须考虑的参量之一。应用牛顿定律可求得试件的最大允许质量。
　　　　F=(M0+M1).A
式中：F为振动台最大正弦振动力(N)
　　　M0为振动台活动部件质量(kg)
　　　M1为试件与夹具的总质量(kg)
　　　A为本次试验的最大加速度(g)
　　例如：以D-1000-5振动台为例。某次试验中，最大加速度为120g,F=10000N，已知该台体活动部件质量为8.3kg，试求：夹具与试件的质量应为多少?
解： ∵F=10000N=1000kg.f　M0=8.3kg
　 由F=(M0+M1).A
得
　　∴最大加速度为120g时，振动台为空载
2.2　最大振动幅值的计算
　　最大振动幅值的计算，主要是根据本次试验参量来确定和检查是否符合振动系统最大特性曲线的允许范围。是本次振动试验能够安全有效地进行的必要技术准备。
2.2.1　在等加速度定振试验中，下限频率点处的位移幅值D和速度幅值V为该试验频段的最大幅值。
　　　　(13)
　　　　(14)
式中：D为下限频率点处的位移幅值(mm)
　　　V为下限频率点处的速度幅值(mm/s)
　　　A为定振加速度幅值(mm/s2)
　　　f为扫频过程中的下限频率(Hz)
C语言计算程序如下：
#include "math.h"
main()
{
　int i,k;
　float A,V,D,M;
　scanf("%d%d%f",&i,&k,&A);
　printf("Hz　　A(g)　　V(cm/s)　　D(mm)　　Mmax(kg)　　＼n");
　for(;i<=k;i++)
　　{D=(A*10000)/((2*3.14*i)*(2*3.14*i));
　　V=(A*1000)/(2*3.14*i);
　　M=1000/A-8.3;
　　printf("%d(Hz)　%6.2f(g)　%6.2f(cm/s)　%6.3f(mm)　%6.2f(Kg)＼n",i,A,V,D,M);
　}
}
　　说明：k和i为定振级A的上下限频率。该程序完成功能为：从下限频率i开始，以步进为1Hz向上扫频至k。程序中8.3为D-1000-5台体活动部件质量。

3　结束语
　　正弦振动试验过程中的物理量计算是振动试验过程中最基本的环节，也是振动试验需要解决的首要问题；它是振动试验曲线设置的主要依据；更重要的是在维护机器设备的使用方面起重要作用。
